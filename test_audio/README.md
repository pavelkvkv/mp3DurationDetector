## Использование

Эти файлы предназначены для тестирования парсера MP3-файлов, включая:
- Определение точной длительности
- Проверку работы с разными конфигурациями
- Валидация с реальными аудиозаписями
- Тестирование граничных случаев (очень короткие/длинные файлы)

## Команды для проверки

Просмотр информации о файле:
```bash
ffprobe -v error -select_streams a:0 -show_entries stream=duration,sample_rate,channels,bit_rate,codec_name <filename>
```

Проверка всех файлов:
```bash
for f in *.mp3; do echo "$f:" && ffprobe -v error -show_entries format=duration "$f"; done
```
